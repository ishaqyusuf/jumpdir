# Endpoints

## Purpose
Track public command entrypoints.

## How To Use
- Update when CLI commands change.

## Commands
- `jumpdir set <paths...>`
- `jumpdir ls`
- `jumpdir rename <current-name-or-path> <new-alias>`
- `jumpdir open <project>`
- `jumpdir . <project>`
- `jumpdir <project> <script> [args...]`
