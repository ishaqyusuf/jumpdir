## Positioning
`jumpdir` is for developers who run local projects from a multi-tabbed terminal and do not want to remember where every repo lives.

## Launch Tweet Thread
Tweet 1:

```text
I got tired of typing cd into random folders just to start local dev servers.

My coding workflow has become more AI + terminal driven, so I’m constantly jumping between repos, running scripts, and opening projects from different tabs.

So I built jumpdir: a tiny CLI for finding and running local repos by name.
```

Tweet 2:

```text
Set your repo folders once:

jumpdir set ~/Documents/code ~/Desktop/projects

Then:

jumpdir ls
jumpdir my-app dev
jumpdir open my-app

No more remembering paths for every side project, client app, or experiment.

GitHub:
https://github.com/ishaqyusuf/jumpdir
```
