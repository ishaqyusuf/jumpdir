# Marketing

## Purpose
Keep public-facing positioning, launch copy, and social posts for `termcode`.

## How To Use
- Add marketing copy here before publishing it.
- Keep language aligned with the project story in `brain/product/vision.md`.
- Avoid overclaiming; focus on the terminal workflow and local repo discovery.

## Positioning
`termcode` is for developers who run local projects from a multi-tabbed terminal and do not want to remember where every repo lives.

## Launch Tweet Thread
Tweet 1:

```text
I got tired of typing cd into random folders just to start local dev servers.

My coding workflow has become more AI + terminal driven, so I’m constantly jumping between repos, running scripts, and opening projects from different tabs.

So I built termcode: a tiny CLI for finding and running local repos by name.
```

Tweet 2:

```text
Set your repo folders once:

termcode set ~/Documents/code ~/Desktop/projects

Then:

termcode ls
termcode my-app dev
termcode open my-app

No more remembering paths for every side project, client app, or experiment.

GitHub:
https://github.com/ishaqyusuf/termcode
```
